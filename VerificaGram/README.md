### Compilando e executando
Para executar o programa deve-se compila-lo:
```
make
```
ou usando o make passando a palavra como argumento:
```
make a=string
```
Ao compilar, execute o arquivo, passando como argumento a palavra a ser buscada:
```
./main string
```
Para deleter os arquivos *.o
```
make clean
```
